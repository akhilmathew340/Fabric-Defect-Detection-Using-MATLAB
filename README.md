# Fabric-Defect-Detection-Using-MATLAB
Image processing-based fabric defect detection using MATLAB for automated textile quality inspection through segmentation and morphological analysis.


## Overview

This project presents an image processing-based Fabric Defect Detection system developed using MATLAB. The system identifies defects in fabric images through a sequence of preprocessing and segmentation techniques, helping automate quality inspection in textile manufacturing.

The project demonstrates the application of digital image processing methods for detecting damaged regions in fabric materials.

---

## Features

- Fabric image preprocessing
- Grayscale image conversion
- Noise reduction using Median Filtering
- Automatic thresholding using Otsu's Method
- Binary image segmentation
- Morphological filtering
- Defect region detection
- Automated quality inspection workflow

---

## Technologies Used

- MATLAB
- Image Processing Toolbox

---

## Project Workflow

1. Load fabric image
2. Convert RGB image to Grayscale
3. Apply Median Filter
4. Perform Otsu Thresholding
5. Convert image into Binary format
6. Remove unwanted noise
7. Detect fabric defects
